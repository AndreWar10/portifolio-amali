# portifolio-amali

Portfólio web da agência **Amáli** — Publicidade e Propaganda, UNIFRAN Franca.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Framer Motion

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Foto hero

Coloque a foto das fundadoras em `public/images/hero.png`. Enquanto o arquivo não existir, o site usa um placeholder em gradiente.

## Deploy (Vercel)

1. Conecte o repositório na [Vercel](https://vercel.com)
2. Deploy automático a cada push na branch principal
